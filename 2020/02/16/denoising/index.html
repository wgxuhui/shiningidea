图像去噪相关论文

  • Image Blind Denoising With Generative Adversarial Network Based Noise Modeling
  • GRDN:Grouped Residual Dense Network for Real Image Denoising and GAN-based Real-world Noise Modeling

1. Image Blind Denoising With Generative Adversarial Network Based Noise Modeling

Chen J, Chen J, Chao H, et al. Image blind denoising with generative adversarial network based noise modeling[C]//Proceedings of the IEEE Conference on Computer Vision and Pattern Recognition. 2018: 3155-3164.
CVPR2018的论文，不到两年时间被引频率快100了，怎么有种……生化环材的感觉？
本文的主要贡献是将GAN网络引入了图像去噪，采用GAN来生成去噪模型，这里注意是只有噪声的图片，而不是带有实际图像的噪声图片，用这种方法来规避GAN网络存在的不稳定性问题以及真假莫辨的问题。我自己感觉文章最重要的创新就是GAN生成单独的噪声图像，也就是用GAN来制造图像对，再送入DCNN进行训练，本身也没有特别大的创新，所以图什么的就不放了。（越来越懒了……）

2. GRDN:Grouped Residual Dense Network for Real Image Denoising and GAN-based Real-world Noise Modeling

Kim D W, Ryun Chung J, Jung S W. Grdn: Grouped residual dense network for real image denoising and gan-based real-world noise modeling[C]//Proceedings of the IEEE Conference on Computer Vision and Pattern Recognition Workshops. 2019: 0-0.
提出了一种grouped residual dense network(GRDN)，我不知道这个应该怎么翻译，是先进、广义的RDN。主要是采用了RDN中的剩余密集块(groups residual density block, GRDB)，作为GRDN的构建模块，这个GRDB块可以显著提高去噪效果。另外还用GAN进行了真实噪声的建模。
文中在背景部分讲了两种噪声建模方式，一种是高斯+泊松噪声，一般是合成噪声训练＋真实图像测试；另一种是直接采集真实场景的数据，但数据对不好获取。文中的方法应该是把高斯+泊松噪声换成了GAN。训练网络就是多个RDB的堆叠。

训练时采用了只使用合成噪声、只使用CHDB的真实噪声数据对、混合使用（ChDB占90%），最终，以极其微弱的优势获胜。废了这么大劲，最后性能提升0.0004- -
